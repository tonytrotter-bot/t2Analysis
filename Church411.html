How to add or replace photos

This page loads photos from the Photos folder next to this HTML file. Each photo is named by the person's full name (e.g., Photos/Ruth Barrett.jpg).

Supported formats: JPG, PNG, JPEG, WebP, and HEIC. The page automatically tries each format, so you can use whichever you have.

To add a missing photo: Save an image into the Photos folder and name it exactly as the person appears here (e.g., Renee Taylor.jpg or Renee Taylor.png). Refresh the page and it will appear.

To replace a photo: Simply overwrite the file in Photos with a new image using the same filename, then refresh.

To adjust a photo: Enter Edit Mode, then click any photo to open the adjuster. Drag to reposition and use the slider to zoom in or out. Your adjustments are saved automatically.

Tip: Square images (same width and height) will look best in the circular frames. Any size will work, but squares won't get cropped unevenly.
